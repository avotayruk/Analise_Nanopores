Осторожно, навайбкодено 

Данные на вход: txt файл с одним столбцом измерений тока в pA
Фацл должен лежать в том же месте, где и программа/приложение

В конце программа сохраняет два файла:

- С названием типа ИМЯФАЙЛА_МЕТОД_ЧАСТОТА_КОЛВО_СОБЫТИЙ.xlsx
  с характеристиками событий (Номер события, Время начала (с), Амплитуда (pA)

- С названием файла ИМЯФАЙЛА_МЕТОД_ЧАСТОТА_raw_±КОЛВОТОЧЕКВОКРУГСОБЫТИЯ.xlsx
  с данными каждого события: ток+время +- event_buffer точек

При необходмости можно скачать csv файл со всеми рассчитанными точками выбранного метода (в нем не выделены события)

Если в папке уже есть файл с таким же названием, то он заменится, изначальный удалится. Если этот файл с тем же названием в той же папке уже открыт, то программа выдаст ошибку, из-за чего следующие файлы не скачаются (потенциально фиксится в будущем)
